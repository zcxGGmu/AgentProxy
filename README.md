# AgentProxy

Directly reuse Coding Agent capabilities instead of reimplementing.

## Development

```bash
pnpm install
pnpm run agentproxy -- --help
pnpm run agentproxy -- --version
pnpm run typecheck
pnpm run test
pnpm run lint
pnpm run format:check
pnpm run build
```

## Scope

AgentProxy v1 is a thin control plane for OpenCode. It does not reimplement an agent runtime.
